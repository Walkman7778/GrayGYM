## ⚙️ Архитектурные решения и вызовы

В процессе разработки этого приложения я столкнулся с несколькими интересными инженерными задачами:

1. **Проектирование БД для результатов игр:** Было необходимо создать эффективную структуру таблиц, которая позволяла бы быстро записывать результаты частых коротких игровых сессий и связывать их с профилем пользователя без потери производительности.
2. **Оптимизация производительности:** В играх важен плавный UI. Я уделил особое внимание оптимизации отрисовки элементов и управлению памятью, а также провел глубокий рефакторинг первоначального кода для улучшения его читаемости и поддержки.
